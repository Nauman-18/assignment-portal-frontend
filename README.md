# Assignment Portal Frontend (React.js)

## Overview
This is the client-side application for the MERN Assignment Workflow Portal, built using React.js and Context API for state management. It features a single login page with role-based redirection to the Student or Teacher Dashboard.

## Technologies Used
* React.js & Vite
* Context API (for state management)

## Setup Instructions

1.  **Install Dependencies:**
    ```bash
    npm install
    # or yarn install
    ```

2.  **Configure API URL:**
    Ensure your API helper (`src/api.js`) points to the correct backend URL (e.g., `http://localhost:5000/api`).

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    The application will typically be available at `http://localhost:5173`.
